## Description

A novel submission site developed by me.

CRUD機能(生成、読み取り、更新、削除の機能)を備えたユーザー登録と縦書き記事（小説）投稿を主軸に作成。また、階層構造（管理者用システム導入）を持たせることで、不適切ユーザーの排除や、エラーの際のバックアップ管理
